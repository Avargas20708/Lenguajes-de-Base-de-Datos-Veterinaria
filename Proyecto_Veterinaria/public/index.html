¿Quiénes somos?

Happy Pet es una clínica veterinaria orientada al bienestar animal. Este sistema permite digitalizar procesos básicos de atención, evitando registros manuales dispersos y facilitando el control de información.

👤 Clientes

CRUD completo para registrar, consultar, editar y eliminar clientes.

🐶 Mascotas

Registro de mascotas asociado a un cliente existente.

📅 Citas

Agenda veterinaria con fecha, hora, mascota y motivo.

🩺 Historial clínico

Registro de diagnóstico y tratamiento por mascota.

🧾 Facturas

Control de facturación asociada a clientes.

🔒 API estable

La conexión se configura por variables de entorno y no queda amarrada a un equipo específico.
